# 💰 Zorvyn Finance Dashboard

A fast, responsive, and professional-grade Finance Management Portal. I built this for the **Zorvyn Frontend Developer Intern** assignment to help users track spending, visualize trends, and manage transactions.

### 🔗 [Live Demo](https://zorvyn-sujay-dashboard.netlify.app/) | [GitHub Repository](https://github.com/sujay-707/zorvyn-finance-dashboard)

---

## 🚀 Key Features

### 1. Financial Overview
* **Smart Cards:** View Total Balance, Income, and Expenses instantly.
* **Live Status:** Includes pulsing "Live Update" indicators for a modern feel.

### 2. Data Visualization (Charts)
* **Spending Trend:** A smooth Line Chart showing how spending changes over time.
* **Expense Breakdown:** A Donut Chart to see which categories take up your budget.

### 3. Smart Analytics & Insights
* **Top Expense:** Automatically identifies your highest spending category.
* **Monthly Comparison:** Shows a percentage growth or decline compared to last month.
* **AI Tips:** Simple logic-based savings recommendations.

### 4. Advanced Transaction Ledger
* **Search & Filter:** Find records by typing any category or description.
* **Smart Sorting:** Click headers to sort by **Date** or **Amount**.
* **Empty State UI:** A professional "No records found" view when searches return zero results.

### 5. Role-Based UI (RBAC)
* **Admin vs. Viewer:** Switch roles using the navbar toggle.
* **Admin:** Full access to manage data.
* **Viewer:** Read-only mode to prevent unauthorized changes.

---

## 🛠️ Tech Stack

* **React.js:** Functional components and modern hooks.
* **Tailwind CSS v4:** High-performance, mobile-first styling.
* **Context API:** Clean global state management (UserStore).
* **Framer Motion:** For premium "slide-up" and "fade-in" animations.
* **Lucide React:** Minimalist, consistent iconography.
* **Recharts:** Interactive, responsive data visualizations.

---

## 🌓 Modern Capabilities

* **Dark Mode:** A professional Slate/Navy theme that is easy on the eyes.
* **Persistence:** Uses `LocalStorage` to remember your data and theme settings after a refresh.
* **Responsive Design:** Works perfectly on Mobile, Tablet, and Desktop with a custom Hamburger menu for small screens.

## ⚙️ Installation & Setup

1.  **Clone the project:**
    ```bash
    git clone [https://github.com/sujay-707/zorvyn-finance-dashboard.git](https://github.com/sujay-707/zorvyn-finance-dashboard.git)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the app:**
    ```bash
    npm run dev
    ```

---

## 🧠 The "Senior-Level" Approach

* **Design:** I used a "Pure White" theme for clarity and high-contrast text for accessibility.
* **Performance:** I used the **Context API** instead of Redux to keep the bundle size small and the logic clean.
* **UX Details:** I focused on "Empty States" and "Sorting" to ensure the app feels like a finished product, not just a demo.
